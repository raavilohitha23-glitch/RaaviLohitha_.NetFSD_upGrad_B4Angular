# Upgrad Virtual Event Management System

##  Project Overview
This is a web-based Event Management System built using **HTML5, CSS3, Bootstrap, and JavaScript**.  
It allows participants to view and register for events, and admins to manage events (add, search, delete).  
The project was developed as part of the Upgrad coursework and follows all requirements in the project document.

##  Features
### Participant
- View upcoming events on the Home page.
- Register for events via the Register page (auto-fills event details).
- Contact the organization using the Contact Us page.

### Admin
- Login with credentials (`admin@upgrad.com / 12345`).
- Add, view, and delete events dynamically.
- Search events by ID, name, or category.
- Logout functionality.

1. Download or clone the project folder.
2. Open `home.html` in your browser.
3. Navigate using the navbar:
   - Home → Upcoming events
   - Login → Admin login
   - Events → Manage events
   - Register → Participant registration
   - Contact Us → Submit queries
